# [Section 2 Daily Coding 🌟] powerOfTwo

### 문제 설명

<p>수를 입력받아 2의 거듭제곱인지 여부를 리턴해야 합니다.</p>

### 입력 / 인자 1 : num

 <ul>
    <li>number 타입의 정수 (num >= 1)</li>
 </ul>

### 출력

 <ul>
    <li>boolean 타입을 리턴해야 합니다.</li>
 </ul>

### 주의 사항

 <ul>
    <li>반복문(while)문을 사용해야 합니다.</li>
    <li>2의 0승은 1입니다.</li>
    <li>Number.isInteger, Math.log2, Math.log 사용은 금지됩니다.</li>
 </ul>
