# [Section 2 Daily Coding 🌟] convertDoubleSpaceToSingle

### 문제 설명

<p>문자열을 입력받아 해당 문자열에 등장하는 두 칸의 공백을 모두 한 칸의 공백으로 바꾼 문자열을 리턴해야 합니다.</p>

### 입력 / 인자 1 : str

 <ul>
    <li>string 타입의 문자열</li>
 </ul>

### 출력

 <ul>
    <li>string 타입의 문자열</li>
 </ul>

### 주의 사항

 <ul>
    <li>두 칸을 초과하는 공백은 존재하지 않는다고 가정합니다.</li>
 </ul>
